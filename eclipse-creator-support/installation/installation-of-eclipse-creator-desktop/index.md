Below is a guide for the installation of the Eclipse Creator desktop app.

On your desktop/laptop, navigate to the webpage [here](https://eclipsecreator.com/fourplus-vr-training-download/):

![Eclipse Creator Download Page](/_images/ec-download-page.png "Eclipse Creator download page")

*If you experience this pop-up, choose to trust and keep the file and report as safe:*

![Eclipse Creator Install](/_images/ec-setup.png "Eclipse Creator setup")

- Run the installation script for Eclipse Creator on desktop.
- Open the Eclipse Creator desktop app.
- Enter the license key:

![Eclipse Creator Licence Entry](/_images/ec-licence-entry.png "Eclipse Creator licene entry")

*This will be contained in an email from a FourPlus representative and be a 24-digit, alphanumeric string. It may be referred to as 'Eclipse Creator Desktop License'. Ensure this is not shared with anyone outside of your institution.*

- Once your license has been validated you can gain access to the pre-installed default collection of SOPs.

*This Default collection will include 6 pre-made SOPs to get users started.*
