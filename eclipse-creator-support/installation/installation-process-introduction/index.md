In order to install Eclipse Creator onto desktop and headset you will require access to a personal or business Meta account and also 2 license keys, which will be supplied to you by a FourPlus representative. The Eclipse Creator VR app is available on the Meta Horizon Store for Meta Quest 2+ devices, whilst the Eclipse Creator Desktop app can be downloaded from the Eclipse Creator website on Mac and Windows platforms.
